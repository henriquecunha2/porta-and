Este projeto tem fins exclusivamente didáticos para aprender como usar o básico das seguintes ferramentas Synopsys:

- vlogan
- vcs
- verdi
- dc_shell

Aos alunos do CI-Expert:

Após fazer clone desse projeto, realizem as seguintes operações para criar um novo repositório de vocês no github com os arquivos que estão aqui:

git remote remove origin

git remote add origin https://github.com/SEU_USUARIO/NOVO_REPOSITORIO.git

git push -u origin main
